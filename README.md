# Touchdown

A single button space game where timing and fuel management are required to successfully land a rocket.

Live Link to Game: https://rushil070.github.io/Onekey/

Link to Code: https://github.com/rushil070/Onekey

- Control a rocket using only the Space bar
- A limited amount of fuel is present which runs out every time the space bar is pressed.
- Land on a moving landing pad while managing gravity and fuel.
- A toggle background button is present which can switch between two backgrounds.
- Fast and slow mode buttons are present that can increase or decrease the speed of the moving base.
- An instruction window is presented before starting the game.
